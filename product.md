# 제품 컨텍스트: 봄날(Bomnal) 쇼핑몰

## 프로젝트 개요
봄날(Bomnal)은 AI 기반 관리 기능을 갖춘 프리미엄 사용자 경험(UX)을 제공하는 Next.js 기반의 현대적인 이커머스 플랫폼입니다.

## 기술 스택
- **프레임워크**: Next.js (App Router)
- **스타일링**: Tailwind CSS (주요), Vanilla CSS (애니메이션/커스텀 속성)
- **데이터베이스**: Firebase (Firestore) / Storage / Auth
- **AI 통합**: Google Gemini (제품 분석, 태그 자동화)
- **언어**: TypeScript

## 아키텍처 원칙
- **컴포넌트 설계**: 아토믹 디자인 원칙, 재사용 가능한 UI 토큰 활용.
- **상태 관리**: Zustand (전역 인증/장바구니 상태 관리).
- **디자인 시스템**: 프리미엄 미학, 생동감 있는 색상, 글래스모피즘(Glassmorphism), 부드러운 애니메이션.

## 현재 집중 분야
- AI 기반 제품 관리 (관리자 기능)
- 사용자 경험 강화 (장바구니, 체크아웃, 검색)
- 안정성 및 성능 최적화
