# 디버그 스쿼드 (Debug Squad)

문제 식별 및 해결에 집중합니다.

## 에이전트
- **Debug_Syntax (문법)**: TypeScript 타입 수정, 정적 분석 오류 해결.
- **Debug_Runtime (런타임)**: 크래시 조사, 예외 처리.
- **Debug_Logic (로직)**: 비즈니스 로직 결함, 계산 오류 수정.
- **Debug_Dependency (의존성)**: 패키지 버전 충돌, 환경 문제 해결.
