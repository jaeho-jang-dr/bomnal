# 프론트엔드 스쿼드 (Frontend Squad)

UI 구축 및 클라이언트 사이드 로직에 집중합니다.

## 에이전트
- **FE_Structure (구조)**: 시맨틱 HTML/JSX 뼈대를 작성합니다.
- **FE_Logic (로직)**: React Hooks, 상태 관리, 비즈니스 로직을 구현합니다.
- **FE_Styler (스타일)**: 디자인 시스템 통합, CSS/Tailwind 스타일링을 담당합니다.
