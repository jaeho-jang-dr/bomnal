# 기획 스쿼드 (Planning Squad)

요구사항을 구체적인 기술 명세와 디자인 및 아키텍처로 변환하는 데 집중합니다.

## 에이전트
- **PM_Requirements (기획자)**: 제품 관리자. 모호한 아이디어를 사용자 스토리(User Story)와 기능 명세서(PRD)로 변환합니다.
- **UI_UX_Designer (디자이너)**: 명세를 컴포넌트 계층 구조와 디자인 토큰으로 시각화합니다.
- **System_Architect (아키텍트)**: 폴더 구조, 데이터 흐름, 기술 스택을 설계합니다.
