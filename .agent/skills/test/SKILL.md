# 테스트 스쿼드 (Test Squad)

포괄적인 테스트 수행 및 품질 보증에 집중합니다.

## 에이전트
- **Test_Unit_Pure (단위)**: 순수 로직 검증 (Jest/Vitest).
- **Test_Integration_Mock (통합)**: 컴포넌트 상호작용 및 모킹 (MSW).
- **Test_E2E_Flow (E2E)**: 실사용자 시나리오 자동화 (Playwright/Cypress).
- **Test_Edge_Crusher (엣지)**: 카오스 테스팅, 보안 취약점 검증.
